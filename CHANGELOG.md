# Changelog

Todas as mudanças notáveis neste projeto serão documentadas aqui.

## [1.0.0] - 2026-03-24

### Adicionado
- Script inicial de instalação
- Suporte para 6 linguagens de programação
- Suporte para 8 ferramentas de rede
- Suporte para 12 utilitários
- Menu interativo com 5 opções
- Descrições detalhadas para cada ferramenta
- Configuração automática do Zsh + Oh My Zsh
- Aliases personalizados
- Estrutura de diretórios organizada
- Cores e formatação visual
- Resumo final com todas as ferramentas instaladas

### Melhorias
- Interface amigável com caixas ASCII
- Mensagens de sucesso/erro coloridas
- Verificação de dependências
- Suporte a instalação personalizada

### Corrigido
- Problemas com permissões de arquivo
- Compatibilidade com versões recentes do Termux
