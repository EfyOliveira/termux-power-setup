## ✨ Características

- 🎯 **Instalação Inteligente** - Escolha o que instalar
- 📚 **Educativo** - Cada ferramenta vem com descrição detalhada
- 🎨 **Design Clean** - Interface amigável e organizada
- ⚡ **Configuração Automática** - Aliases, Zsh e Oh My Zsh
- 📁 **Estrutura Organizada** - Diretórios prontos para projetos
- 🔧 **Ferramentas Essenciais** - Linguagens, rede e utilitários

## 🚀 Instalação Rápida

```bash
# Clone o repositório
git clone https://github.com/EfyOliveira/termux-power-setup.git

# Entre no diretório
cd termux-power-setup

# Dê permissão de execução
chmod +x setup.sh

# Execute o script
./setup.sh
